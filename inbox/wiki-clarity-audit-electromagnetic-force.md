# Wiki明瞭さ監査: electromagnetic-force（電磁力）

## 問題点

### 視覚的表現
- [SVG全般] 3次元的な力の方向が2Dで表現されており、空間認識が必要な学習者には不十分
- [フレミング図] B・I・Fの色分けが統一されていない（赤=力・青=磁界・緑=電流 の一貫適用が理想）
- [左手則説明] 左手（電動機用）と右手（発電機用）の比較図がなく、混同しやすい
- [垂直性の明示] B・I・Fが互いに垂直である関係を示す直角記号が不足

### 計算式の網羅性
- [公式セクション] `F = BIl` の基本式のみで、**角度依存 `F = BIl sinθ`** が記載されていない
  - 電験3種では角度ありの問題が頻出。この欠落は誤答の直接原因になる

### 引っかけポイントの内容
- [引っかけポイント] 左手則と右手則の混同について言及があるが、FBI記憶法と手の向きの対応が明示されていない
- [引っかけポイント] 実効長さ（B・Iに垂直な成分のみ）の説明が薄い
- [引っかけポイント] コイル内の各辺に別々に法則を適用する必要性（複数方向の力）への言及がない
- [単位] T（テスラ）とその他磁界単位の混同リスクへの注意書きがない

### 構造・配置
- 現状は引っかけポイントが公式の直後にある（今回修正済み）← OK
- アンペアの法則の適用条件（幾何学的要件）の説明が抽象的

## 改善案

### 優先度A（電験合格に直結）

1. **[公式セクション] 角度依存式の追加**
   ```
   F = BIl sinθ
   （θ: 電流方向と磁界方向のなす角。θ=90°のとき最大）
   ```
   - 「なぜsinθか」：B・Iが平行（θ=0°）のとき力はゼロ。直角（θ=90°）のとき最大。
   - 例題で「45°の場合」を1問追加

2. **[引っかけポイント] FBI記憶法の明確化**
   ```
   F（中指=Force/力）- B（人差し指=磁界）- I（親指=電流）
   ※ 左手のみ有効（電動機・モータ）
   右手は発電機（フレミングの右手則）→ 別ページ参照
   ```

3. **[引っかけポイント] 実効長さの罠を追加**
   > 斜め配置の導体は「磁界に垂直な長さの成分」のみが力に寄与する。全長lをそのまま使うのは誤り。

### 優先度B（理解深化）

4. **[SVG] 色コード統一**
   - 力（F）→ 赤矢印
   - 磁界（B）→ 青矢印
   - 電流（I）→ 緑矢印
   - 全SVGに適用し、凡例をSVG内に記載

5. **[比較図] 左手則 vs 右手則 並列表示**
   - 「電動機=左手」「発電機=右手」を横並びで比較するSVGを追加
   - 現状この比較がなく、混同リスクが高い

6. **[コイル問題] 複数方向の力の扱い**
   > コイルの各辺に対して個別にフレミングの法則を適用する必要がある。
   > 例：矩形コイルでは左辺・右辺で力の向きが逆になり、回転トルクが生じる。

### 優先度C（補足）

7. **[単位] T（テスラ）の注記**
   - `1 T = 1 Wb/m² = 10,000 Gauss`の変換を補足（計算問題での単位ミス対策）

8. **[アンペアの法則] 適用条件の明示**
   - 対称性が必要（無限長直線・ソレノイド・トロイダルコイル等）
   - 「すべての電流配置に使えるわけではない」を強調

## 実装順序

```
1. F = BIl sinθ 追加（公式セクション）← 最優先
2. FBI記憶法の明確化（引っかけポイント）
3. 実効長さの罠（引っかけポイント）
4. SVG色コード統一
5. 左手・右手則比較SVG追加
6. コイル複数方向の力（例題または説明追加）
```
