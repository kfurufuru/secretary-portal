## 3. SetsuchiIchiranPage（接地工事一覧）

### 計算式の有無
**あり** — 以下の箇所に計算式が含まれる：

1. ConclusionBox: `B種: 変圧器中性点 → 150/Ig Ω / 4.0 mm 以上`
2. MemTable rows: `"150 / Ig"` セル
3. note: `"B種の 150/Ig は「1秒以内に自動遮断される場合は 300/Ig」に緩和される"`
4. QuickReview: `{ q: "B種接地工事の接地抵抗値の計算式は？", a: "150 / Ig（Ig: 1線地絡電流）" }`

### C項目チェック結果

| 項目 | 結果 |
|---|---|
| C1: 導出過程または「なぜその係数か」の根拠1行 | **不足** — 150の根拠（系統電圧・地絡電流との関係）が説明されていない |
| C2: 公式の物理的意味が説明されているか | **不足** — Ig（1線地絡電流）の定義は括弧内に記載されているが、「なぜ150を割るのか」の意味がない |
| C3: 単位整合性 | OK — 150 [V/(A)] ÷ Ig [A] = Ω の単位整合は取れている |
| C4: 試験範囲外の計算の混入 | なし — 300/Igへの緩和条件は試験頻出事項。適切な範囲 |

### 重大 / 中度 / 軽微
**中度**

### 不動判定: **差し戻し（C1・C2 要補足）**

B種の150という係数は「低圧電路の最大使用電圧（150V系）を地絡電流で割った接触電圧制限値に由来する」という物理的背景がある。丸暗記させるだけなら私の監修を通す意味がない。QuickReviewの直前か、ConclusionBoxのnoteに1行でよい。例：「150の根拠：低圧電路の対地電圧上限（約150V）を地絡電流Igで割った制限値」。300/Igへの緩和条件の記述は試験頻出であり削除不要。

## 4. ZetsuenIchiranPage（絶縁耐力試験一覧）

### 計算式の有無
**あり** — 以下の箇所に倍率を用いた計算が暗示されている：

1. ConclusionBox: `低圧（7kV以下）: 1.5倍 × 10分`、`高圧（7kV超 60kV以下）: 1.25倍 × 10分`、`特高（中性点直接接地、170kV超）: 0.64倍 × 10分`
2. MemTable: 倍率列に 1.5倍 / 1.25倍 / 0.64倍 の3値
3. GoalQuestion・QuickReview・TrapTable 全体で倍率数値を繰り返し使用

※ 明示的な計算式（V × 倍率 = 試験電圧V の形）は記述されていないが、倍率を用いて試験電圧を求めるB問題構造が前提となっている。

### C項目チェック結果

| 項目 | 結果 |
|---|---|
| C1: 導出過程または「なぜその係数か」の根拠1行 | **不足** — 1.5倍・1.25倍・0.64倍の物理的根拠が全くない |
| C2: 公式の物理的意味が説明されているか | **不足** — 「倍率」の意味（何に対して何倍か）は文脈で分かるが、係数の出処が不明 |
| C3: 単位整合性 | OK — 最大使用電圧[V] × 倍率[無次元] = 試験電圧[V]。問題なし |
| C4: 試験範囲外の計算の混入 | なし — 試験電圧の倍率計算は電験法規B問題の核心。適切 |

### 重大 / 中度 / 軽微
**中度**

### 不動判定: **差し戻し（C1 要補足）**

1.5倍と1.25倍の差、そして0.64倍という特異な数値について説明がない。ここを素通りするとB問題で「なぜこの倍率か」と問われた変化球に対応できない。少なくとも以下を1行で補足せよ：

- **1.5倍（低圧）の根拠**: 電技解釈第16条の「最大使用電圧の1.5倍の電圧で10分間耐えること」。低圧は絶縁余裕を大きく取る設計思想。
- **1.25倍（高圧・一般特高）の根拠**: 電技解釈第16条。高圧以上は系統管理が厳格なため余裕を1.25倍に設定。
- **0.64倍（中性点直接接地・特高）の根拠**: 中性点を直接接地することで対地電圧が相電圧（線間電圧の1/√3 ≈ 0.577倍）に抑制されるため、試験電圧も通常より低く設定できる。0.64倍は1.1 × (1/√3) ≈ 0.635 を丸めた実用値。

「これは何条のどの計算式ですか？」と問われたら答えられる状態にしてから公開すること。
