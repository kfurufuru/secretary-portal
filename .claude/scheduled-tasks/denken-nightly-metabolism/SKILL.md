# Denken Nightly Metabolism v0（最小版）

## 設計哲学（3者レビューを反映）

- **落合**: 自動生成は「思考の代替」にならず「思考の起動」に限定する → 出力は**問いのみ**、答えは翌朝バスで古舘さんが書く
- **ホリエモン**: 最小スコープ。Sonnet一発、1日1ファイル
- **ひろゆき**: e-logスキーマを固定（確認済み：テーブル形式 `| 日付 | 科目 | 問題 | 誤答理由 | 正しい考え方 |`）

## スコープ（v0）

**やること**:
- 毎晩22:00、`denken-study/e-log.md` の「誤答メモ」テーブルから**当日追記分**を抽出
- Sonnet一発で「自問自答用の問い」を3つだけ生成
- `inbox/denken-questions/YYYY-MM-DD.md` に出力

**やらないこと**（v0意図的カット）:
- 因果解説の自動生成（人間が書く）
- knowledge/ への自動書き込み（knowledge-rules.mdの人間確認前提ルール遵守）
- 健全性スコアリング
- Notion連携

## 入出力

**入力**: `denken-study/e-log.md` の「## 誤答メモ」セクション当日行のみ

**空入力時の挙動**: 当日追記ゼロなら何もせず `logs/denken-nightly.log` に `no-op` と記録して終了

**出力**: `inbox/denken-questions/YYYY-MM-DD.md`

```markdown
---
type: denken-question
date: YYYY-MM-DD
source: e-log.md当日分
status: pending
---

# 明日のバスで考える3問

## 1. [問題文から派生した問い]
（※答えを書く欄。空欄のまま通勤中に記入）

## 2. ...

## 3. ...

---
元の誤答（参考）:
- 日付 / 科目 / 問題 / 誤答理由 / 正しい考え方
```

## Sonnetプロンプト（固定）

```
あなたは電験3種学習支援AIです。古舘さん（36歳・電気エンジニア管理職・合理主義）の誤答メモから、
翌朝のバス通勤中（15分・スマホ・紙ペンなし）で考えられる「問い」を3つ生成してください。

## 絶対ルール
- 語呂合わせ・暗記テクニック禁止（因果理解ベース）
- 答えは書かない。問いだけ
- 1問30秒で考えられる粒度
- 「なぜ」「どちらが先」「もし〜なら」形式推奨

## 入力
（当日の誤答メモテーブル行）

## 出力
問い1: ...
問い2: ...
問い3: ...
```
