Business Skills — Overview

成果を出す人の
スキル地図

「話がうまい」「知識が多い」は成果の根拠にならない。ビジネスで差がつく能力を4つの柱・7つの核心スキルに分解する。

結論：問題を見抜き、整理し、伝え、人を動かして、結果まで持っていく力
01 — 4つの柱
考える・動かす・伝える・整える

スキルは孤立していない。4つの柱が連動して初めて成果になる。いずれかが欠けると、残りの能力が活かされない。

考える力
Thinking

仕事の本質は作業をこなすことではなく、「何が問題か」「何を優先すべきか」を見抜くこと。成果を出す人は問題の背後にある構造を見ている。

課題設定力 — 本当の問題を見つける
論理的思考力 — 原因と結果を整理する
構造把握力 — 複雑な状況を全体像でつかむ
意思決定力 — 情報不十分でも判断し進める
動かす力
Execution

考えるだけでは成果にならない。ビジネスでは「良い案」より実際に形にしたものの方が価値を持つ。周囲を巻き込みながら前に進める力が必要。

実行力 — 決めたことを具体的な行動に落とす
推進力 — 曖昧な状況でも停滞させず進める
段取り力 — 順序・時間・関係者を整理する
やり切る力 — 結果が出るまで持っていく
伝える力
Communication

どれだけ良い考えでも、伝わらなければ存在しないのと同じ。相手が納得して動ける状態をつくることが重要であり、「正しいこと」をするだけでは足りない。

説明力 — 相手に合わせてわかりやすく話す
文章力 — 要点を短く誤解なく伝える
傾聴力 — 相手の意図・本音を正確に受け取る
交渉力 — 利害が違う相手と合意点を見つける
整える力
System

継続的に成果を出す人は、能力だけでなく仕事の進め方そのものを整える。毎回ゼロから頑張る人は消耗する。強い人は努力を仕組みに変える。

優先順位付け — 重要度と緊急度を見極める
時間管理 — 限られた時間を成果に変える
情報整理力 — 蓄積・検索・再利用できる形に
再現性づくり — 属人化を防ぎ仕組みに落とす
02 — 特に重要な7つ
実務で差がつくコアスキル

全16スキルのうち、「何をすべきかを見抜き、周囲を動かし、形にする」ために特に差がつきやすい7つ。

CORE 01
課題設定力
目の前の現象ではなく本当の問題を見つける。問いが正しければ解の半分は見えている。
CORE 02
論理的思考力
原因と結果を整理し筋道立てて考える。感覚・経験だけに頼らず、構造で判断する。
CORE 03
構造把握力
複雑な状況を分解し全体像をつかむ。正しい支点に力をかけるために必要な技術。
CORE 04
実行力
決めたことを具体的な行動に落とす。アイデアは実行して初めて価値になる。
CORE 05
段取り力
順序・時間・関係者を整理して進行する。段取り8割、実行2割が成果の分岐点。
CORE 06
説明力
相手に合わせてわかりやすく伝える。結論先・根拠付き・相手視点が3原則。
CORE 07
優先順位付け
重要度と緊急度を見極め集中する。すべてに全力は「何もしていない」と同義。
03 — 習熟段階
初級・中級・上級で変わるスキル

求められるスキルはキャリアステージで変化する。現在地を確認し、次の段階を設計する。

初級
基礎実行力
報連相（報告・連絡・相談）
基本的な文章力
タスク管理（ToDo・期限）
指示を正確に実行する力
中級
自律推進力
問題解決力（5Why・MECE）
関係者調整力
プレゼン力（構造化説明）
改善提案力（課題→施策→効果）
上級
設計・再現力
課題設定力（問いを作る）
意思決定力（不確実性下）
組織を動かす力
仕組み化・再現性構築力
AI時代にさらに重要になるスキル
04 — AI時代の再定義

AIが普及すると単純作業の価値は下がる。逆説的に、次の力はむしろ希少価値が高まる。AIにできないのは「判断・問い・文脈・本質・共感」だからだ。

問いを立てる力
判断する力
文脈を理解する力
本質を見抜く力
人を動かす力
これからは「作業者」としての能力より、「設計者・判断者」としての能力が重要になる。AI活用が上手い人とは、AIに正しい問いを与え、判断の最終責任を担える人のことだ。
ひとことで言うと
問題を見抜き、整理し、伝え、
人を動かして、結果まで持っていく力
