# パワーエレクトロニクス

## TL;DR
機械科目で毎年2〜3問。**三相整流回路の出力電圧計算とサイリスタの動作原理**が最頻出。インバータ・チョッパは穴埋め型。電力変換の「直流↔交流↔直流」の方向感を押さえる。

## 1. 整流回路

### 単相半波整流
```
Vd = Vm/π ≈ 0.318 Vm   （Vm: 最大値）
```

### 単相全波整流（ブリッジ）
```
Vd = 2Vm/π ≈ 0.637 Vm
```

### 三相全波整流（最頻出）
```
Vd = (3√6/π) × V_ph ≈ 2.34 × V_ph   （V_ph: 相電圧実効値）
Vd = (3√2/π) × V_L ≈ 1.35 × V_L     （V_L: 線間電圧実効値）
```

> [!key-insight] 三相全波整流の係数「1.35」は頻繁に出る。導出: 三相交流のピーク電圧 √2×V_L を6分割して平均を取ると 3√2/π × V_L。直感的には「ピーク電圧の約95%を直流で取り出せる」。

### 位相制御（サイリスタ整流）
```
Vd = Vd0 × cosα   （α: 点弧角）
```
- α=0° のとき最大直流電圧 Vd0
- α を大きくすると出力電圧低下 → 速度制御に利用

## 2. 直流チョッパ

### 降圧チョッパ（Buck）
```
Vo = D × Vi   （D = ton/(ton+toff): デューティ比）
```

### 昇圧チョッパ（Boost）
```
Vo = Vi / (1-D)
```

> [!key-insight] 昇圧チョッパの出力が入力より高くなる理由: スイッチオフ時にインダクタが蓄積エネルギーを放出し、入力電圧に重畳するから。エネルギー保存則。D→1に近づくほど出力は∞に向かう（実際は損失で制限）。

## 3. インバータ（DC→AC変換）

- **電圧形インバータ**: 直流電圧源からPWMでスイッチング → 交流波形を生成
- **PWM（パルス幅変調）**: キャリア周波数と信号波の比較でスイッチングタイミングを決定
- 高調波低減のためスイッチング周波数を高くする

### 用途
- **可変速ドライブ（VFD）**: 三相誘導電動機の可変速制御 = 現代の標準
- 太陽光発電の系統連系インバータ

## 4. サイリスタ（SCR）

- **ターンオン**: ゲートにパルス電流を印加（一度オンになるとゲート制御不要）
- **ターンオフ**: 電流がゼロになる瞬間（転流）でオフ
- **逆阻止3端子サイリスタ**: A・K・G の3端子

> [!gap] サイリスタは「ゲートでオフできない」（GTO以外）。オフには電流を強制的に遮断するか、交流の電流ゼロ点を使う（自然転流）。問題文で「ゲートオフ」という表現が出たら誤りの選択肢。

## 5. 電力変換の分類

| 変換 | 装置 | 用途 |
|------|------|------|
| AC→DC | 整流器（コンバータ） | 充電・直流電源 |
| DC→DC | チョッパ | 直流電圧変換 |
| DC→AC | インバータ | モータ駆動・系統連系 |
| AC→AC | サイクロコンバータ | 可変周波数変換 |
