# 電子理論

## TL;DR
理論の中では比較的特殊な分野。**バイポーラトランジスタの増幅回路・演算増幅器・MOSFETの空欄穴埋め**が主体。半導体の物理より「回路の動作」を重視する。

## 頻出テーマ（過去11年分より）

### 1. バイポーラトランジスタ増幅回路

```
電流増幅率: hFE = IC / IB   (直流電流利得)
電流利得: Ai = -hfe/(1 + hoe×RL)  ≈ -hfe  (交流小信号)
電圧利得: Av = -hfe×RL / hie
```

- エミッタ接地回路: 電流増幅・電圧増幅・位相反転あり（最もよく出る）
- ベース接地・コレクタ接地（エミッタフォロワ）の特徴も把握

> [!key-insight] h パラメータは4つあるが電験3種では hie（入力インピーダンス）と hfe（電流増幅率）の2つが実質的に問題に出る。hre, hoe は省略可として扱う問題が多い。

### 2. 演算増幅器（Op-Amp）

**反転増幅回路**:
```
Av = -Rf / Rin    (負号は位相反転を意味する)
```

**非反転増幅回路**:
```
Av = 1 + Rf / Rin
```

**加算回路・差動増幅回路**:
- 加算: Vo = -(Rf/R₁×V₁ + Rf/R₂×V₂)
- 差動: Vo = (Rf/R)(V₂ - V₁)（R₁=R₂=R, Rf=帰還抵抗の場合）

> [!key-insight] 理想Op-Ampの「仮想短絡（バーチャルショート）」原理: 負帰還があれば V⁺ ≈ V⁻ が成立する。この原理から計算を始めると全ての回路が解ける。

### 3. FET（電界効果トランジスタ）

#### 3-1. FETの分類と「なぜ電圧制御か」

FETはゲートに電圧をかけるだけでドレイン電流（ID）を制御する。**バイポーラが電流（IB）で制御するのに対し、FETはゲート電流がほぼゼロ**。なぜか → ゲート絶縁膜（MOSFETの場合は酸化膜）があるため、DC電流が物理的に流れない。

| 種類 | 構造の違い | ゲート電流 | 典型的な用途 |
|------|----------|-----------|------------|
| JFET（接合型） | pn接合で空乏層制御 | ほぼゼロ（逆バイアス漏れ電流のみ） | 低ノイズ入力段 |
| MOSFET（MOS型） | 酸化膜でゲート絶縁 | 完全にゼロ（絶縁体） | スイッチング・パワー制御 |

> [!note] 電験では「FET = 電圧制御素子・高入力インピーダンス」がまず問われる。バイポーラとの対比で選択肢を絞る問題が多い。

#### 3-2. MOSFETの2タイプ — なぜ特性が真逆か

**エンハンスメント型（Enhancement）**
- VGS = 0 ではチャネルなし → ID = 0（ノーマリーオフ）
- VGS > Vth（閾値電圧）を超えると絶縁膜下に反転層（チャネル）が誘導される → 電流が流れ始める
- 「エンハンス（強化）」＝ゲート電圧でチャネルを作り出す、と覚えるより「何も印加しなければオフ」という動作から理解する

**ディプレッション型（Depletion）**
- VGS = 0 でも元からチャネルが存在 → IDが流れる（ノーマリーオン）
- 負の VGS でチャネルを狭めてカットオフ（JFETに近い動作）
- 「ディプレッション（空乏化）」＝ゲート電圧でチャネルを削る

> [!tip] 電験の選択肢で「VGS < 0 でオフになる」はディプレッション型、「VGS > 0（Vth以上）でオンになる」はエンハンスメント型と即判定できる。

#### 3-3. ID-VDS特性曲線（過去問で毎回グラフが出る）

特性曲線の領域分けを理解することが最重要。

```
       ID
        |        飽和領域
        |  ___________________  VGS = 5V
        | /
        |/___________________   VGS = 3V
        |
        |______              VGS = Vth（ギリギリオン）
        +---------------------> VDS
         線形領域 | 飽和領域
                 ↑ピンチオフ点
```

**線形領域（VDS < VGS − Vth）**
- IDはVDSに比例して増加する
- FETをスイッチとして使うとき（オン状態）はここで動作させる

**飽和領域（VDS ≥ VGS − Vth）**
- IDはVDSが増えても増加しない（電流が飽和）
- なぜか → ドレイン側でもチャネルがピンチオフされ、それ以上VDSを上げても電子の流れ（速度）が制限されるから
- **増幅回路はこの飽和領域で動作させる**

> [!warning] ピンチオフ＝「電流がゼロ」ではない。チャネルが消えてもドリフト電流でIDは流れ続ける。電験の引っかけ選択肢に注意。

#### 3-4. 相互コンダクタンス gm（電圧→電流の変換率）

**gm = ΔID / ΔVGS（VDS一定のとき）**

- 単位はA/V（またはS：ジーメンス）
- 「VGSを1V変えたときIDが何A変わるか」 = 増幅回路の利得を決める数値
- gmが大きい ＝ 同じVGS変化で大きなID変化 = 高利得

> [!note] 過去問では ID-VGS 特性グラフの傾きから gm を読み取る問題が出る。グラフの傾き ΔID/ΔVGS がそのまま gm。

#### 3-5. ソース接地増幅回路（最重要回路）

バイポーラのエミッタ接地に相当する基本増幅回路。

**電圧利得 Av = −gm × RD**

- マイナス符号 = 位相反転（入力が上がると出力が下がる）
- RDはドレイン抵抗
- バイポーラの Av = −hfe×RL/hie と形が似ているが、FETは入力電流がゼロなので分母に入力インピーダンスが出てこない

> [!tip] 計算問題のパターン: gmとRDが与えられて「電圧利得を求めよ」。掛け算一発。gmはグラフから読むか問題文に与えられる。

#### 3-6. バイポーラ vs FET — 電験頻出の比較表

| 比較項目 | バイポーラ（BJT） | FET（MOSFET） |
|---------|----------------|--------------|
| 制御方式 | 電流制御（IB→IC） | 電圧制御（VGS→ID） |
| 入力インピーダンス | 低〜中（数百Ω〜数kΩ） | 超高（数MΩ〜GΩ超） |
| ゲート/ベース電流 | 必要（IB≠0） | 不要（≈0） |
| 熱ノイズ | 少ない | やや多い |
| 集積化（IC） | ○ | ◎（LSI・パワー回路の主役） |
| スイッチング速度 | 速い | 高周波帯ではMOSの方が有利な場面も |

> [!key-insight] 電験の論説問題で「FETはなぜ高入力インピーダンスか」→「ゲートが絶縁膜で覆われており電流が流れないから」が正解の核心。

### 4. ダイオード・整流回路

- pn接合ダイオードの特性（順方向電圧降下 ≈ 0.7V）
- **バラクタダイオード（可変容量ダイオード）**: 逆バイアス電圧でpn接合の空乏層幅が変化 → 静電容量が変化。FM変調・同調回路に使用
- 太陽電池: 光起電力効果（フォトボルタイック効果）

### 5. 半導体基礎

- n型: ドナー（P, As, Sb等）添加 → 電子が多数キャリア
- p型: アクセプタ（B, Al等）添加 → 正孔が多数キャリア
- フェルミ準位の位置（n型は伝導帯寄り、p型は価電子帯寄り）

## 落とし穴

> [!warning] 演算増幅器の「仮想短絡」はあくまで負帰還がある場合にのみ成立。開ループ（帰還なし）では成立しない。問題文の回路構成を確認する。

> [!warning] FET — ピンチオフ点を超えても電流がゼロになるわけではない。飽和領域ではIDがほぼ一定（≠0）で流れている。「ピンチオフ＝電流遮断」は誤り。

> [!warning] FET — JFETはVGSを必ず逆バイアス（nチャネルなら負）で使う。MOSFETエンハンスメント型はVGS > Vthが必要（正の電圧で動作）。符号を混同しやすい。

> [!warning] FET — ディプレッション型の「ノーマリーオン」はスイッチ用途では危険（電源を入れただけで電流が流れる）。エンハンスメント型（ノーマリーオフ）が安全側で使いやすいため電力用途に主流。この安全性の理由まで説明できると論説問題で差がつく。

## 過去問出題パターン（FET）

| パターン | 何を見ているか | 対策 |
|---------|-------------|------|
| ID-VDS グラフ読み取り → 動作領域を判定 | 飽和/線形の境界理解 | VDS < VGS−Vth が線形、それ以上が飽和と即判定 |
| ID-VGS グラフの傾き → gm を読む | gm = ΔID/ΔVGS | グラフから2点取って差分計算 |
| ソース接地回路の電圧利得 | Av = −gm×RD の適用 | gmとRDを掛けてマイナスをつける |
| 選択肢: 「FETの特徴として正しいものは？」 | 電圧制御・高Zin・ゲート電流ゼロ | この3点セットで正答を選ぶ |
| エンハンス/ディプレッション型の識別 | VGS=0でオンかオフか | ノーマリーオフ=エンハンス、ノーマリーオン=ディプレッション |
