# 電子理論

## TL;DR
理論の中では比較的特殊な分野。**バイポーラトランジスタの増幅回路・演算増幅器・MOSFETの空欄穴埋め**が主体。半導体の物理より「回路の動作」を重視する。

## 頻出テーマ（過去11年分より）

### 1. バイポーラトランジスタ増幅回路

```
電流増幅率: hFE = IC / IB   (直流電流利得)
電流利得: Ai = -hfe/(1 + hoe×RL)  ≈ -hfe  (交流小信号)
電圧利得: Av = -hfe×RL / hie
```

- エミッタ接地回路: 電流増幅・電圧増幅・位相反転あり（最もよく出る）
- ベース接地・コレクタ接地（エミッタフォロワ）の特徴も把握

> [!key-insight] h パラメータは4つあるが電験3種では hie（入力インピーダンス）と hfe（電流増幅率）の2つが実質的に問題に出る。hre, hoe は省略可として扱う問題が多い。

### 2. 演算増幅器（Op-Amp）

**反転増幅回路**:
```
Av = -Rf / Rin    (負号は位相反転を意味する)
```

**非反転増幅回路**:
```
Av = 1 + Rf / Rin
```

**加算回路・差動増幅回路**:
- 加算: Vo = -(Rf/R₁×V₁ + Rf/R₂×V₂)
- 差動: Vo = (Rf/R)(V₂ - V₁)（R₁=R₂=R, Rf=帰還抵抗の場合）

> [!key-insight] 理想Op-Ampの「仮想短絡（バーチャルショート）」原理: 負帰還があれば V⁺ ≈ V⁻ が成立する。この原理から計算を始めると全ての回路が解ける。

### 3. MOSFET（電界効果トランジスタ）

- **ドレイン電流**: ゲート・ソース間電圧 VGS で制御（電圧制御素子）
- エンハンスメント型: VGS > Vth でオン
- ディプレッション型: VGS = 0 でオン、負のVGSでオフ
- バイポーラと違い、ゲート電流が（ほぼ）流れない → 高入力インピーダンス

### 4. ダイオード・整流回路

- pn接合ダイオードの特性（順方向電圧降下 ≈ 0.7V）
- **バラクタダイオード（可変容量ダイオード）**: 逆バイアス電圧でpn接合の空乏層幅が変化 → 静電容量が変化。FM変調・同調回路に使用
- 太陽電池: 光起電力効果（フォトボルタイック効果）

### 5. 半導体基礎

- n型: ドナー（P, As, Sb等）添加 → 電子が多数キャリア
- p型: アクセプタ（B, Al等）添加 → 正孔が多数キャリア
- フェルミ準位の位置（n型は伝導帯寄り、p型は価電子帯寄り）

## 落とし穴
> [!gap] 演算増幅器の「仮想短絡」はあくまで負帰還がある場合にのみ成立。開ループ（帰還なし）では成立しない。問題文の回路構成を確認する。
