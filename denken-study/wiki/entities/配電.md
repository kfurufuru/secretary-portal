# 配電

## TL;DR
電力科目で毎年2〜3問。**電圧降下・電力損失の計算と力率改善コンデンサの容量**が最頻出。単相3線式の中性線電流と不平衡電圧も必出。送電線路と公式は共通だが「低圧配電の現場」が題材になる。

## 1. 配電線の電圧降下・電力損失

### 三相3線式
```
ΔV = √3 × I × (R cosφ + X sinφ)   [V]  （線間電圧降下）
ΔP = 3I²R = P²R / V²               [W]  （3線分の電力損失）
```

### 単相2線式
```
ΔV = 2I(R cosφ + X sinφ)           [V]  （往復2線分 → 係数2）
ΔP = 2I²R                          [W]
```

> [!key-insight] 単相2線式は係数が「2」。往復の2本分の電圧降下・損失を計算する。三相3線式の係数「√3」と混同しやすい。

### 単相3線式（頻出）
```
上線電流 I1, 下線電流 I2 のとき:
中性線電流 In = I1 - I2   （不平衡時）
平衡時(I1=I2): In = 0   （中性線に電流なし）
```
- 上線・下線の電圧は各々 100V、線間電圧 200V
- 不平衡時の各電圧は中性線インピーダンスで計算

## 2. 力率改善（コンデンサ容量）

```
Q_c = P × (tanθ₁ - tanθ₂)   [kvar]

P:  有効電力 [kW]
θ₁: 改善前の力率角（cos θ₁ = 改善前力率）
θ₂: 改善後の力率角
```

**コンデンサ設置による効果:**
- 電流減少 → 電圧降下減少・電力損失減少
- 供給電力増加（設備余裕の有効活用）

> [!key-insight] 力率改善は「電流を減らす」ことが本質。有効電力は変わらず無効電流を打ち消す。損失はI²R∝I²なので電流が半分になると損失は1/4になる。

## 3. 配電方式の比較

| 方式 | 特徴 | 信頼性 | 損失 |
|------|------|--------|------|
| 樹枝状（放射状） | 単純・安価 | 低（停電範囲大） | 大 |
| ループ（環状） | 切替で停電縮小 | 中 | 中 |
| スポットネットワーク | 並列供給・無停電 | 高 | 小 |

## 5. 地中ケーブル

- **充電電流**: I_c = 2πf × C × V （C: 対地静電容量、V: 相電圧）
- 地中ケーブルは架空線より静電容量が大きい → 充電電流が大きい → 長距離送電に不利
- **誘電損**: P_d = ω × C × V² × tan δ （tan δ: 誘電正接）

## 落とし穴
> [!gap] 電圧降下の公式に使う電圧は「受電端電圧」か「送電端電圧」かで計算値が変わる。近似計算では「定格電圧」を使うことが多いが、問題文を確認する。また「相電圧」か「線間電圧」かも常に確認。
