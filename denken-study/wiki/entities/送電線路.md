# 送電線路

## TL;DR
電力科目の最大分野（毎年3〜5問）。**電圧降下・電力損失・短絡電流・弛度**の計算が超頻出。三相交流の√3と電圧の定義（線間か相電圧か）に常に注意する。

## 1. 電圧降下・電力損失

### 三相送電線（近似式）
```
ΔV = √3 × I × (R cosφ + X sinφ)   [V]  (線間電圧降下)
ΔV = I × (R cosφ + X sinφ)          [V]  (相電圧降下 ×√3で線間)
ΔP = 3 × I² × R = P² × R / V²     [W]  (3線分の電力損失)
```

> [!key-insight] 電圧降下は「抵抗降下×cosφ + リアクタンス降下×sinφ」の形。力率1（cosφ=1）のとき純抵抗降下だけ残る。送電電圧が高いほど電流が減り損失は電圧の2乗に反比例して激減。

## 2. 架空送電線の弛度（たるみ）

```
D = WS² / (8T)    [m]

W: 電線の単位長さあたりの重量 [N/m]
S: 径間（スパン） [m]
T: 電線の水平張力 [N]
```

- 電線長さ: L = S + 8D²/(3S) ≈ S（Dが小さい場合）
- **弛度が大きすぎる** → 地表・他の線・樹木への接触リスク
- **弛度が小さすぎる（張力過大）** → 低温時の断線リスク

## 3. 1線地絡・3相短絡電流

### 対称座標法の結果（重要公式）
```
1線地絡電流: I_g = 3E / (Z1 + Z2 + Z0)

3相短絡電流: I_s = E / Z1
```
- Z1: 正相インピーダンス
- Z2: 逆相インピーダンス
- Z0: 零相インピーダンス

> [!key-insight] 非接地系（中性点非接地）ではZ0が大きく→地絡電流が小さい。直接接地系ではZ0が小さく→地絡電流大・保護継電器が確実に動作。接地方式の選択は地絡保護と過電圧のトレードオフ。

## 4. 電力系統の安定度

```
P = (VS × VR / X) × sinδ   （最大送電電力はδ=90°のとき）
P_max = VS × VR / X
```

- **定態安定度**: 定常状態での安定性
- **過渡安定度**: 事故除去後の安定性（等面積法で判定）
- 安定化対策: 高速遮断・自動再閉路・直列コンデンサ

## 5. 保護継電方式

| 継電器 | 用途 |
|--------|------|
| 過電流継電器（OCR） | 過負荷・短絡保護 |
| 距離継電器 | 送電線の短絡・地絡保護（インピーダンスで距離測定）|
| 地絡方向継電器（DGR） | 高感度地絡検出・方向判別 |
| 差動継電器 | 変圧器・母線の内部故障 |

## 落とし穴
> [!gap] 電圧降下の公式で「線間電圧」か「相電圧」かを間違えると√3倍の誤差が出る。問題文の電圧表記（kV単位が線間電圧のことが多い）を確認する。
