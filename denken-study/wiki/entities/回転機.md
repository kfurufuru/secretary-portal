# 回転機

## TL;DR
機械科目の核心。毎年5〜7問出題。**誘導電動機のすべり・トルク計算が最頻出**。同期機のV字曲線と直流機の速度制御も必出。「なぜこのトルク式か」を回路と力の関係から理解する。

## 1. 誘導電動機（最頻出）

### すべり s
```
s = (Ns - N) / Ns
Ns = 120f / p   [rpm]   (同期速度, p:極数, f:周波数)
```

### 等価回路とトルク
```
二次入力 P2 = T × ωs        (ωs = 2πNs/60)
二次銅損 Pc2 = s × P2
機械出力 Pm = (1-s) × P2
効率    η = Pm / P1 ≈ (1-s)   (無負荷損無視の場合)
```

> [!key-insight] 「二次入力：二次銅損：機械出力 = 1 : s : (1-s)」の比が超重要。トルク・効率・すべりの全計算はここから展開できる。

### 始動法
| 方法 | 始動電流 | 始動トルク | 用途 |
|------|---------|-----------|------|
| 全電圧始動 | 定格の5〜7倍 | 大 | 小容量 |
| Y-Δ始動 | 全電圧の1/3 | 全電圧の1/3 | 中容量 |
| リアクトル始動 | 低減 | 低減 | 中・大容量 |
| インバータ始動 | 低電流で大トルク | 優秀 | 現代主流 |

### 速度制御
- **周波数制御（インバータ）**: Ns ∝ f → 現代の主流
- **極数変換**: 段階的速度変更
- **二次抵抗制御**: すべり増加→効率低下（巻線形のみ）

## 2. 同期機

### 電機子反作用
- **遅れ電流（誘導性負荷）**: 減磁作用 → 端子電圧低下
- **進み電流（容量性負荷）**: 増磁作用 → 端子電圧上昇
- **純抵抗負荷**: 交差磁化作用

### V字曲線（同期電動機）
```
界磁電流 If ↑ → 力率進み（進相運転）= コンデンサ代わり
界�力電流 If ↓ → 力率遅れ（遅相運転）
最小電機子電流点が力率1
```

> [!key-insight] 同期電動機を「同期調相機」として運転すると無効電力を制御できる。系統の力率改善に使う現場知識と直結。

### 短絡比 Ks
```
Ks = 無負荷定格電圧の界磁電流 / 三相短絡定格電流の界磁電流
    = 1 / 同期インピーダンス（%値）
```
- Ks大 → 安定度高・機械大型・高価

## 3. 直流機

### 発電機の電圧方程式
```
E = V + IaRa   (分巻・他励発電機, Ia:電機子電流)
E = k × φ × n  (k:機械定数, φ:磁束, n:回転速度)
```

### 電動機の速度制御
```
N ∝ (V - IaRa) / φ
```
- **界磁制御**: φ↓ → N↑（定出力制御）
- **電圧制御**: V↑ → N↑（ワード・レオナード方式）
- **抵抗制御**: 効率悪い（過去の方式）

## 落とし穴
> [!gap] 誘導電動機の「二次入力＝トルク×同期角速度」と「機械出力＝トルク×実回転角速度」の違い。同期速度と実速度を区別しないと分子分母が逆になる。
