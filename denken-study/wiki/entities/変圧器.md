# 変圧器

## TL;DR
機械・法規・電力の3科目横断で出題。**%インピーダンス降下と電圧変動率、最大効率条件**が最頻出。等価回路を描いてから計算する習慣を作る。

## 1. 等価回路と試験

### 無負荷試験（開放試験）
- 二次側開放 → 一次側に定格電圧印加
- 測定: 鉄損 Pi [W]、励磁電流 I0、鉄損抵抗 r0、励磁リアクタンス x0

### 短絡試験
- 二次側短絡 → 一次側に低電圧（定格電流が流れる電圧）印加
- 測定: 定格電流時の銅損 Pc [W]、インピーダンス電圧 Vz

## 2. 百分率インピーダンス降下（%Z）

```
%Z = Vz / V1n × 100 [%]
%R = Pc / Pn × 100  (Pn:定格容量)
%X = √(%Z² - %R²)
```

> [!key-insight] %Zが大きいと短絡電流が小さくなる（安全だが電圧変動が大きい）。系統の短絡容量計算に直結。

## 3. 電圧変動率

```
ε = %R×cosφ + %X×sinφ   [%]   (近似式・遅れ力率)
```

- cosφ = 1（抵抗負荷）のとき: ε ≈ %R
- sinφ = 1（純誘導負荷）のとき: ε ≈ %X

## 4. 効率と最大効率

```
η = P_出力 / (P_出力 + Pi + Pc×m²)   (m=負荷率=I/In)
最大効率条件: Pi = Pc×m²  → m = √(Pi/Pc)
```

> [!key-insight] 最大効率は「鉄損＝銅損」の時。Pi は負荷に無関係（一定）、Pc は負荷の2乗に比例（m²×Pc）。なぜ？鉄損は電圧で決まり、銅損は電流²×抵抗だから。

## 5. 並行運転の条件
1. 巻数比（変圧比）が等しい
2. 極性が同じ
3. %Zが等しい（異なると負荷分担が容量比に反比例）
4. 周波数・波形が同じ（交流変圧器）

> [!gap] 並行運転で%Zが異なると、%Zの小さい変圧器に電流が多く流れる。問題では「%Zが小さい=電流多め」で計算する。直感と逆に感じやすい（抵抗が小さい方に電流→当然）。

## 6. 全日効率（法規科目と重複）
```
η_d = ΣP_出力 × t / (ΣP_出力 × t + Pi×24 + ΣPc×m²×t)
```
