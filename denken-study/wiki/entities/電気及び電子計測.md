# 電気及び電子計測

## TL;DR
出題数は少ない（1〜2問）が確実に得点したい分野。**2電力計法・各種計器の動作原理・誤差の種類**がほぼ毎年出る。

## 頻出テーマ

### 1. 2電力計法（三相電力測定）

```
P = W₁ + W₂          (有効電力)
Q = √3 × (W₁ - W₂)   (無効電力)
力率 cos θ = (W₁ + W₂) / (√3 × V × I)
```

> [!key-insight] W₁ と W₂ の差が無効電力に比例する。力率1のとき W₁ = W₂。力率0のとき W₁ = -W₂（一方が負）。

### 2. 各種計器の動作原理

| 計器 | 原理 | 用途 |
|------|------|------|
| 可動コイル形 | 永久磁石磁界と電流の力 | 直流のみ |
| 可動鉄片形 | 電流による鉄片の磁気力 | 交直両用（商用周波数） |
| 電流力計形 | 固定・可動コイル間の力 | 交直両用・高精度電力計 |
| 熱電形 | ジュール熱で熱電対発電 | 高周波測定可能 |
| 整流形 | 整流→可動コイル | 交流用（実効値を示す） |

### 3. 測定誤差

- **相対誤差（百分率誤差）**: (測定値 - 真値) / 真値 × 100 [%]
- **感度**: 指針の振れ / 入力量の変化
- 系統誤差・偶然誤差・過失誤差の区別

### 4. オシロスコープ
- 電圧の時間波形を観測
- 波形から周波数・周期・位相差・実効値（正弦波なら Vm/√2）を読み取る
- リサジュー図形から位相差・周波数比を判定
