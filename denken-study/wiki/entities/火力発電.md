# 火力発電

## TL;DR
毎年2〜3問。**熱効率計算と汽力サイクルの構成**が必出。コンバインドサイクルの効率向上の理由を因果で説明できると応用が利く。

## 1. 熱効率

```
η_送 = P_送電 / (mf × Hf)     [—]
η_発 = P_発電端 / (mf × Hf)

mf: 燃料消費量 [kg/h or m³/h]
Hf: 燃料の発熱量 [kJ/kg or kJ/m³]
```

> [!key-insight] 「送電端熱効率 < 発電端熱効率」。送電端は変圧器・送電線損失が引かれる。問題で「送電端か発電端か」を必ず確認する。

## 2. 汽力（ランキン）サイクル

```
ボイラ（熱追加）→ タービン（仕事）→ 復水器（熱放出）→ 給水ポンプ（昇圧）→ ボイラ
```

- **再熱サイクル**: タービン中段で再加熱 → 熱効率向上・湿り蒸気防止
- **再生サイクル**: 抽気で給水加熱 → 熱効率向上（ボイラへの給水温度上昇）
- 最終給水温度が高いほど熱効率が良い（カルノー効率の上限温度上昇）

## 3. コンバインドサイクル（高効率）

```
ガスタービン（1000〜1500℃）→ 排熱回収ボイラ → 蒸気タービン
総合熱効率: 50〜60%（汽力単独35〜40%より大幅向上）
```

> [!key-insight] 効率が高い理由: カルノー効率 η = 1 - T_低/T_高 において、T_高（ガスタービン入口温度）が蒸気タービン単独より高いため。高温の排ガスを捨てずに二重利用。

## 4. 各種設備

| 設備 | 機能 |
|------|------|
| ボイラ | 燃料燃焼→蒸気生成 |
| 蒸気タービン | 蒸気の熱エネルギー→回転力 |
| 復水器 | タービン排気蒸気→水に戻す（真空維持が重要） |
| 給水加熱器 | 抽気で給水を予熱（再生サイクル） |
| 脱気器 | 給水中の溶存酸素除去（腐食防止） |

## 5. 環境対策設備

- **電気集じん装置**: コロナ放電でフライアッシュを捕集
- **排煙脱硫装置**: SOx（硫黄酸化物）を石灰石でCaSO4に変換
- **排煙脱硝装置**: NOxをアンモニアで触媒分解（N2+H2O）
- **低NOxバーナー**: 燃焼温度を下げてサーマルNOx低減

## 落とし穴
> [!gap] 熱効率の分母は「投入熱量（燃料の発熱量）」であり、「燃料消費量」ではない。単位変換（kJ/kg × kg/h = kJ/h）が必要。問題の単位を必ず確認する。
