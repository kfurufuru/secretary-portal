# 自動制御

## TL;DR
毎年1〜2問。**ブロック線図の等価変換と閉ループ伝達関数の導出**が核心。PID各動作の特性は「P＝比例、I＝残留偏差除去、D＝応答高速化」で因果理解する。

## 1. ブロック線図と伝達関数

### 閉ループ伝達関数（フィードバック制御）
```
G_cl = G / (1 + GH)

G: 前向き伝達関数（制御要素）
H: 帰還伝達関数（センサ等）
```

### 等価変換のルール
- 直列: G1 × G2
- 並列: G1 + G2
- フィードバック: G / (1 + GH)
- 引き出し点・加算点の移動は符号に注意

## 2. PID制御

| 動作 | 伝達関数 | 効果 | 欠点 |
|------|---------|------|------|
| P（比例） | Kp | ゲイン調整・応答速い | 定常偏差が残る |
| I（積分） | Ki/s | 定常偏差を除去 | 応答遅延・振動しやすい |
| D（微分） | Kds | 急変に即応・偏差変化率に反応 | ノイズ増幅 |

> [!key-insight] I動作が「定常偏差を除去する」理由: 偏差が残り続けると積分値が増え続け、ついに出力が偏差を消すまで動く。定常状態では偏差=0でないとI出力が安定しない。

## 3. 一次遅れ要素

```
G(s) = K / (1 + Ts)   (T:時定数, K:ゲイン)
```
- ステップ応答: y(t) = K(1 - e^(-t/T))
- 時定数T後: 最終値の約63.2%に到達

## 4. 安定判別

- **ラウス・フルビッツ安定判別**: 特性方程式の係数から安定性を判定
- **ナイキスト線図**: 開ループ伝達関数の周波数応答から安定余裕を評価
- **ゲイン余裕・位相余裕**: 安定性の指標（通常ゲイン余裕＞6dB、位相余裕＞45°）
