# 電気回路

## TL;DR
理論科目の最大分野（毎年7〜9問）。**直流回路の定理・RLC交流・三相交流・過渡現象**が4大テーマ。ブリッジ回路の平衡条件とテブナンの定理は毎年どちらかが出る。

## 頻出テーマ（過去11年分より）

### 1. 直流回路の定理

**キルヒホッフの法則**:
- 電流則（KCL）: ノードに流入する電流の和 = 流出の和
- 電圧則（KVL）: 閉ループの電圧降下の和 = 起電力の和

**テブナン・ノートンの定理**:
- テブナン: 複雑な回路 → 電圧源 Vth + 直列抵抗 Rth に等価変換
- 最大電力: 負荷抵抗 R_L = Rth のとき最大電力 P_max = Vth²/(4Rth)

> [!key-insight] 最大電力問題は「テブナン等価回路を作れるか」が全て。開放電圧と短絡電流を求める手順を確立する。

**ブリッジ回路の平衡条件**:
```
R₁/R₂ = R₃/R₄    （対辺の積が等しい: R₁×R₄ = R₂×R₃）
```

### 2. 交流回路（インピーダンス・電力）

```
Z_R = R
Z_L = jωL
Z_C = 1/(jωC) = -j/(ωC)
S = P + jQ = VI*    (複素電力)
P = VI cosθ = I²R   (有効電力)
Q = VI sinθ = I²X   (無効電力)
力率 = cos θ = P/S
```

> [!key-insight] 複素インピーダンスで回路を解く。コンデンサのインピーダンスは周波数が高いと小さい（容量性が支配的）。インダクタは逆。

### 3. RLC直列・並列共振

```
共振周波数: f₀ = 1/(2π√LC)
直列共振: |Z| 最小、電流最大、Q = ω₀L/R
並列共振: |Z| 最大、電流最小
```

> [!gap] 「直列共振で電流最大」「並列共振で電流最小」を逆に覚えやすい。直列: 共振でXL + XC = 0 → Z = R のみ → 電流最大。並列: 共振でYL + YC = 0 → アドミタンス最小 → インピーダンス最大 → 電流最小。

### 4. 三相交流（超頻出）

**Y接続（星形）**:
```
線間電圧 = √3 × 相電圧
線電流 = 相電流
P = √3 × V_L × I_L × cosθ = 3 × V_p × I_p × cosθ
```

**Δ接続（三角）**:
```
線間電圧 = 相電圧
線電流 = √3 × 相電流
```

> [!key-insight] Y-Δ変換: Δの各辺をY等価に変換 → Z_Y = Z_Δ/3。不平衡三相回路は対称座標法が有効だが電験3種では平衡三相が基本。

### 5. 過渡現象

```
RC直列: v_C(t) = V(1 - e^(-t/τ)),  τ = RC
RL直列: i(t) = V/R × (1 - e^(-t/τ)),  τ = L/R
```

- 時定数 τ: 初期値の約63.2%に達する時間
- スイッチ切り替え問題: 切り替え前の初期値を確認してから解く

### 6. 交流電力と力率

- **2電力計法**: P = W₁ + W₂、力率 cos θ = (W₁ + W₂)/(√3 × V × I)

## 落とし穴
> [!gap] RC回路の過渡現象でコンデンサ電圧は「急変しない」（電流は急変できる）。RL回路では電流が急変しない（電圧は急変できる）。スイッチ操作直後の初期値設定に使う。
