# 水力発電

## TL;DR
毎年2問。**出力計算公式とベルヌーイの定理**が必出。水車の種類（ペルトン・フランシス・プロペラ）の選定基準も頻出。計算より概念穴埋め問題が多い。

## 1. 水力発電出力（超頻出）

```
P = 9.8 × Q × H × η   [kW]
P = ρgQH × η          [W]   (ρ=1000 kg/m³, g=9.8 m/s²)

Q: 流量 [m³/s]
H: 有効落差 [m]
η: 総合効率（水車効率×発電機効率）
```

> [!key-insight] 係数9.8（≈g）は「1m³の水が1m落ちると9.8kJ のエネルギー」から来る。水の重さ ρg = 9800 N/m³ × Q[m³/s] × H[m] = 9800×Q×H [W] = 9.8QH [kW]。

## 2. ベルヌーイの定理

```
p/ρg + v²/2g + z = 一定
```
- p: 圧力 [Pa]
- v: 流速 [m/s]
- z: 位置水頭 [m]
- 連続の式と組み合わせた管路計算で出題

## 3. 水車の種類と選定

| 水車 | 適合落差 | 原理 |
|------|---------|------|
| ペルトン水車 | 高落差（300m以上） | 衝動式（ジェット水流が羽根に当たる） |
| フランシス水車 | 中落差（30〜700m） | 反動式（渦巻き状に流入、混流型） |
| プロペラ水車 | 低落差（5〜80m） | 反動式（軸流型、可変翼=カプラン） |
| カプラン水車 | 低落差 | プロペラの改良型（翼角可変→部分負荷効率高） |

> [!key-insight] ペルトンは高落差・少流量。フランシスは中落差・汎用。プロペラ/カプランは低落差・大流量。「落差が高い=水圧が高い=少量の水でもエネルギー大」という物理から選定基準を導ける。

## 4. 揚水発電

- 夜間余剰電力で揚水 → 昼間ピーク時に発電（電力貯蔵の役割）
- 揚水効率 × 発電効率 = 総合効率（通常70〜75%）
- 揚水電力: P_揚 = 9.8×Q×H / η_揚

## 5. 流況曲線と発電計画

- 流量が多い時期（梅雨・台風）と少ない時期で出力変動
- ダム式発電で貯水 → 均等化
- 調整池式: 日負荷変動を吸収（夜間貯水・昼間放水）

## 落とし穴
> [!gap] 「有効落差」は総落差から水路損失・吸出し管損失を引いたもの。問題文に「有効落差」と書いてあれば損失分はすでに考慮済みなので二重に引かない。
