# 電磁気

## TL;DR
理論科目で毎年3〜5問出題。**クーロンの法則・平行板コンデンサ・電磁誘導・ループ電流の力**が頻出4テーマ。公式の「なぜ」を理解すれば計算パターンはシンプル。

## 頻出テーマ（過去11年分より）

### 1. クーロンの法則・電界

```
F = k × Q₁Q₂ / r²    (k = 1/(4πε₀) ≈ 9×10⁹ N·m²/C²)
E = k × Q / r²        (点電荷による電界）
```

- 電界の重ね合わせ: ベクトル合成
- 等電位面: 電界に対して垂直

> [!key-insight] 真空の誘電率 ε₀ = 8.85×10⁻¹² F/m。比誘電率 εr を掛けると媒質内の誘電率 ε = εr × ε₀。コンデンサ容量 C = εS/d に直結。

### 2. 平行板コンデンサ（超頻出）

```
C = ε × S / d           (S:面積, d:極板間距離)
W = ½CV² = Q²/(2C)      (静電エネルギー)
```

- 電荷一定で間隔を変えた場合 vs 電圧一定で変えた場合で計算が違う
- 誘電体を挿入する問題: C が変化 → V または Q が変化

> [!gap] 「電荷固定」か「電圧固定」かを問題文で必ず確認。電荷固定: Q=const → V = Q/C が変化。電圧固定: V=const → Q = CV が変化。エネルギーの変化方向が逆になる。

### 3. 磁界・磁束密度

```
H = I / (2πr)            (直線電流による磁界)
B = μ₀H = μ₀I/(2πr)
F = BIl × sinθ           (ローレンツ力、電流に働く力)
F/l = μ₀I₁I₂/(2πr)       (平行電流間の力)
```

### 4. 電磁誘導（ファラデーの法則）

```
e = -N × dΦ/dt           (誘導起電力)
e = Blv                   (導体が磁界中を運動する場合)
```

- レンツの法則: 誘導電流は磁束の変化を妨げる方向
- 自己インダクタンス L: e = -L × di/dt

### 5. ループ状導体・磁気回路
- 環状コア（トロイダルコイル）の磁界: H = NI / (2πr)
- 磁気回路のオームの法則: F = NI（起磁力）, Rm（磁気抵抗）

## 落とし穴
> [!gap] 電界 E と磁界 H の単位を混同しない。E [V/m], H [A/m], B [T=Wb/m²]。力の公式で B と H を間違えやすい（力の計算は B を使う）。
