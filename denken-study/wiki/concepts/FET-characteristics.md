# FET — ID-VDS特性曲線とピンチオフ

## TL;DR
ID-VDS 特性グラフは「線形領域・飽和領域・ピンチオフ点」の3要素を読めれば解ける。増幅回路は飽和領域で動作させる。ピンチオフ ≠ 電流ゼロ。

## 特性グラフの構造

VGSをパラメータとして複数の曲線が描かれる。VGSが大きいほど上の曲線（IDが大きい）。

```
       ID ↑
          |        ─── ─── ───  VGS = 5V（飽和電流が最大）
          |      /
          |   ─── ─── ───       VGS = 3V
          |  /
          |─── ─── ───          VGS = Vth（最下限の動作曲線）
          +──────────────────→ VDS
           ↑線形    ↑飽和
           ピンチオフ境界
```

## 線形領域 — スイッチのオン状態

**条件: VDS < VGS − Vth**

- IDはVDSに比例して増加する（抵抗のような振る舞い）
- パワーMOSFETのスイッチングはこの領域のオン抵抗（RDS(on)）が重要指標
- デジタル回路のスイッチ動作は線形領域（完全オン時）

## 飽和領域 — 増幅回路の動作点

**条件: VDS ≥ VGS − Vth**

- IDはVDSが増えてもほぼ一定になる
- なぜか → ドレイン側でチャネルがピンチオフされ、電子の流れが電界によって加速・飽和するため
- 増幅回路はこの領域に動作点（Q点）を設定する

> [!warning] 「飽和領域」はバイポーラの「飽和領域」とは意味が全く逆。バイポーラの飽和はスイッチオン状態、FETの飽和は増幅動作領域。同じ言葉で逆の意味なので混同注意。

## ピンチオフとは何か

ピンチオフ = VGS − VDS = Vth の条件でチャネルがドレイン側で消滅する現象。

- チャネルが消えても電流はゼロにならない
- ドレイン電界で電子はドリフト輸送され、IDは飽和電流値を維持し続ける
- 「ピンチオフ ＝ IDがゼロ」は誤りで電験の引っかけ選択肢に使われる

> [!note] ピンチオフ点を超えると ID の VDS 依存性がほぼなくなり、VGS だけで ID が決まる。これが「電圧制御電流源」としてのFETの本質。

## VGSの変化が特性曲線に与える影響

| VGSの変化 | 特性曲線の変化 | IDの変化 |
|----------|-------------|---------|
| VGS を上げる | 曲線が上に移動 | 飽和電流が増加 |
| VGS を下げる | 曲線が下に移動 | 飽和電流が減少 |
| VGS = Vth | 最下限の曲線 | ほぼゼロ（カットオフ寸前） |
| VGS < Vth | チャネルなし | ID = 0（カットオフ） |

## JFETの特性との違い

JFETではVGSは逆バイアス（nチャネルなら負の値）を使う。ピンチオフ電圧 VP は負の値で表される。エンハンスメント型MOSFETとは VGSの符号が逆になるため、グラフの横軸の向きが違って見える。

## 出題パターン（電験3種 理論）

| 出題タイプ | 読み取り内容 | 正解の導き方 |
|-----------|-----------|-----------|
| 動作領域の判定 | VGS・VDS の値から領域を特定 | VDS と VGS−Vth を比較 |
| 飽和電流の読み取り | グラフ上の水平部分のID値 | VGSに対応する曲線の水平部を読む |
| ピンチオフ境界 | VGS=3VのときピンチオフになるVDSは？ | VDS = VGS − Vth を計算 |
| バイポーラとの用語混同を問う | 「飽和領域」の意味 | FETとバイポーラで逆であることを指摘 |
