# FET — エンハンスメント型 vs ディプレッション型 / バイポーラとの比較

## TL;DR
「VGS=0でオンかオフか」だけ見ればエンハンス/ディプレッションを即識別できる。バイポーラとの比較は「電流制御 vs 電圧制御・入力インピーダンス・ゲート電流の有無」の3点で整理する。

## エンハンスメント型 vs ディプレッション型

| 識別点 | エンハンスメント型 | ディプレッション型 |
|-------|----------------|----------------|
| VGS = 0 の状態 | チャネルなし → オフ（ノーマリーオフ） | チャネルあり → オン（ノーマリーオン） |
| オンにする条件 | VGS > Vth（正の閾値を超える） | VGS = 0 でもオン。負にするとオフ |
| チャネルの形成 | ゲート電圧で誘導（誘起）する | もとから存在する。ゲートで空乏化 |
| 安全性 | 電源投入でいきなり電流が流れない | 電源を入れるだけで電流が流れる危険 |
| 主な用途 | パワーMOSFET・デジタルIC | 低雑音アナログ回路（ただし少数） |

> [!note] 電験の識別問題: 「VGS = 0 でドレイン電流が流れる」 → ディプレッション型確定。「ある正のVGS以上で電流が流れ始める」 → エンハンスメント型確定。

## なぜエンハンスメント型が主流か

ノーマリーオフは安全側（フェイルセーフ）。電源を入れただけで電流が流れるノーマリーオン（ディプレッション型）はパワー用途では危険で採用が難しい。デジタル回路でもノーマリーオフの方が消費電力設計がしやすい。

> [!tip] 電験の論説問題で「ノーマリーオフが好まれる理由」を問われた場合: 安全側動作・低消費電力設計・誤動作リスク低減 が正解の核心。

## JFETとMOSFETの根本的な違い

JFETはゲートとチャネルをpn接合で隔てる。MOSFETは酸化膜（絶縁体）で隔てる。

| 特性 | JFET | MOSFET |
|-----|------|--------|
| ゲート絶縁 | pn接合（逆バイアス） | 酸化膜（完全絶縁） |
| ゲート電流 | 極めて小さい漏れ電流がある | 完全にゼロ（絶縁体） |
| VGSの向き（nch） | 負（逆バイアス必須） | 正（エンハンス型）または両方向（ディプレッション型） |
| 種類 | ディプレッション型のみ | エンハンス・ディプレッション両方あり |

## バイポーラ（BJT）とFETの比較

電験で最も頻繁に選択肢で問われる比較。

| 比較項目 | バイポーラ（BJT） | FET（MOSFET） |
|---------|----------------|--------------|
| 制御方式 | **電流制御**（IBがICを制御） | **電圧制御**（VGSがIDを制御） |
| 制御端子への入力 | ベース電流 IB が必要 | ゲート電流は不要（≈0） |
| 入力インピーダンス | 低〜中（kΩオーダー） | 超高（MΩ〜GΩ） |
| 熱ノイズ | 少ない（低雑音） | やや多い |
| 温度特性 | 温度上昇でICが増加（熱暴走リスク） | 温度上昇でIDが減少（自己安定） |
| 集積度 | pnpとnpnの相補が必要（面積大） | CMOSで高集積可能（現代ICの主流） |

> [!key-insight] 「FETが高入力インピーダンスである理由」: ゲートが絶縁膜（酸化膜）で覆われておりDC電流が物理的に流れない。バイポーラはベース電流が必ずある。この物理的な差が入力インピーダンスの桁違いの差を生む。

## 温度特性の自己安定（電験で出る理由）

FETは温度が上がるとキャリア移動度が低下 → ID が減少 → 発熱が抑制される（負帰還）。バイポーラは逆で、温度→IC増加→さらに発熱の正帰還（熱暴走）が起きうる。パワー用途でFETが選ばれる重要な理由の一つ。

## 出題パターン（電験3種 理論）

| 出題タイプ | 正解の核心 | 引っかけポイント |
|-----------|----------|--------------|
| 「FETの特徴として正しいものは？」 | 電圧制御・高Zin・ゲート電流≈0 | 「電流制御」「低Zin」を混在させた選択肢 |
| エンハンス/ディプレッション識別 | VGS=0でオン/オフ | 「VGS<0でオフ」はディプレッション型 |
| バイポーラとの比較 | 制御方式・Zin・ゲート電流の3点 | 「ノイズが少ない」はバイポーラ（FETではない） |
| 温度特性の比較 | FETは自己安定・バイポーラは熱暴走 | 「温度に強い」がどちらかを正確に理解 |

## セルフチェック

??? question "VGS = 0 で ID = 10mA 流れている FET はどちらの型か？"
    ディプレッション型。VGS=0でオン（ノーマリーオン）のため。

??? question "「FETは電流制御素子である」は正しいか？"
    誤り。FETは電圧制御素子。電流制御はバイポーラ（BJT）の特徴。

??? question "FETのゲート電流がゼロに近い理由は？"
    ゲートが酸化膜（絶縁体）で覆われているため、DC電流が物理的に流れない。
