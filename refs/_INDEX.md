## 参照優先順位

1. **第一**: 上記ローカルPDF を `Read` ツールで参照（タイムアウトなし・最新版確実）
2. **第二**: e-Gov API `https://laws.e-gov.go.jp/api/1/lawdata/409M50000400052`（電技省令本体）
3. **第三**: METI 公式サイト（Claude Code環境からは現在TCP接続不可・HTTP 000）

## 運用ルール

- 法令改正の通知があったら、ユーザーが上記ディレクトリのPDFを最新版に差し替える
- 江間ペルソナは Read ツールで該当PDFを読み、別表番号・条文番号・数値を抽出
- 出典セルには取得日ではなく **「電技解釈第○条 別表第○（令和7年11月版）」** と版数で記録
- 数値抽出時は必ず単位（A, MΩ, 秒, kV）と条件（周囲温度・絶縁種別・電線本数等）を併記

## METI 直接DL について

Claude Code環境からは **METI ドメインへのTCP接続が確立しない**（HTTP 000、curl/PowerShell共通）。
原因: サンドボックスのネットワークポリシー or METI側のIPブロック。
→ **DL試行は無意味**。常にローカルPDFを使う。

ブラウザ経由では取得可能なので、改正版が出たらブラウザでDLしてから上記ディレクトリに配置する。
