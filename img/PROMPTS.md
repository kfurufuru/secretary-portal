# img/ SVG生成プロンプト集

## 使い方

1. 下記プロンプトをそのままClaudeに投げる
2. 生成されたSVGを `img/{topic}/{name}.svg` に保存
3. `img/{topic}/SOURCES.md` の該当行を記入する

## SVG共通仕様

```
viewBox:     "0 0 320 200"（全インラインSVGも同じ）
背景:        白前提（imgタグで background:"white" を付ける）
フォント:    sans-serif, font-size="11"
```

### カラーパレット（この6色のみ使う）

| 用途 | カラーコード |
|------|------------|
| ワイヤ・枠線・軸 | `#1e293b` |
| 部品アウトライン（抵抗・コンデンサ） | `#475569` |
| 入力信号・能動部品 | `#7c3aed` |
| 出力信号・動作点 | `#f59e0b` |
| 補足テキスト・軸ラベル | `#64748b` |
| 強調領域（ONゾーン等） | `#16a34a` |

### 余白・レイアウトルール（プロンプトに必ず含める）

- 部品ラベルはrect上辺から **10px以上** 上に配置（上端ギリギリ禁止）
- 出力ラベル（Vo等）はviewBox右端から **25px以上** 余白を取る
- 入力ラベル（Vi等）はviewBox左端から **5px以上** 余白を取る
- 数式ラベルはviewBox下端から **10px以上** 余白を取る（y ≤ 190 に配置）
- 全要素はviewBox内に収める（はみ出し禁止）
- Op-amp三角形ラベル: **上側入力=反転(−)、下側入力=非反転(+)**（逆にしない）

### テキスト重なり防止ルール（生成後に必ず確認）

- **同一 x 軸上の複数テキストは y 差を font-size + 4px 以上** 確保する（例: font-size=11 なら y 差 ≥ 15px）
- **SVG図形（矢印先端・polygon）とテキストが同じ y 座標になる場合はテキストを上下どちらかに 10px 以上ずらす**
- **対角線（line）上にラベルを配置しない**：ラベルは線の端点から垂直方向に 10px 以上オフセットした外側に置く
- **VCC/VEE 等の電源端子を描く場合、数式エリア（最下部2行）と重ならないことを確認**。概念図では電源端子は省略してよい
- 生成後チェックリスト: ①同一x付近の全テキストy値を列挙し最小y差を確認 ②斜め線のラベルが線から離れているか ③下部数式2行がviewBox内に完全収容されているか

## プロンプト一覧

### #op-amp-01｜反転増幅回路

```
以下の仕様でSVGを生成してください。

目的: 演算増幅器（反転増幅）の回路図
仕様:
  viewBox="0 0 320 200" xmlns="http://www.w3.org/2000/svg"
  白背景前提。フォント sans-serif font-size="11"
  カラー: ワイヤ=#1e293b / 部品=#475569 / 入力=#7c3aed / 出力=#f59e0b / テキスト=#64748b

描くべき要素:
  1. 演算増幅器（三角形シンボル、左辺に−/+端子、右頂点に出力）
  2. 入力抵抗 Ri（入力Vi〜反転入力間）
  3. 帰還抵抗 Rf（出力〜反転入力間のループ）
  4. 非反転入力(+)をGNDに接続（三線グランドシンボル）
  5. 入力ノードVi（紫の点）・出力ノードVo（黄の点）
  6. 図の下部に "Av = −Rf / Ri" のラベル

注意:
  - 部品はIEC矩形スタイル（zigzagなし）
  - 部品ラベルは矩形の上に小さく
  - 帰還ループは上を通るルートで描く
```

### #op-amp-02｜非反転増幅回路

```
以下の仕様でSVGを生成してください。

目的: 演算増幅器（非反転増幅）の回路図
仕様:
  viewBox="0 0 320 200" 白背景 カラーは #op-amp-01 と同じ

描くべき要素:
  1. 演算増幅器（三角形シンボル）
  2. 入力Vi → 非反転入力(+)に直接接続
  3. 反転入力(−)と出力間に Rf
  4. 反転入力(−)とGND間に R1
  5. 出力ノードVo（黄）
  6. 下部に "Av = 1 + Rf/R1" ラベル
```

### #op-amp-03｜ボルテージフォロワ（バッファ）

```
目的: ボルテージフォロワ回路図
仕様: viewBox="0 0 320 200" 同カラー

描くべき要素:
  1. 演算増幅器
  2. 入力Vi → 非反転入力(+)
  3. 出力から反転入力(−)へ直接帰還（Rfなし）
  4. 出力Vo
  5. 下部に "Av = 1（バッファ）" ラベル
```

### #semiconductor-01｜PN接合ダイオード（順逆バイアス）

```
目的: PN接合ダイオードの順バイアス・逆バイアス状態の概念図
仕様: viewBox="0 0 320 200" 同カラー

描くべき要素:
  左半分(順バイアス): P|N構造、空乏層縮小、電流矢印
  右半分(逆バイアス): P|N構造、空乏層拡大、電流なし
  各部にラベル（順方向電流・逆飽和電流 IS）
```

### #semiconductor-02｜整流ブリッジ回路

```
目的: 4素子ブリッジ整流回路
仕様: viewBox="0 0 320 200" 同カラー

描くべき要素:
  1. ダイオード4本をひし形配置
  2. AC入力（左右）、DC出力（上下）
  3. 電流経路を矢印で示す
  4. 出力部の+/−ラベル
```

## 新規プロンプト追加手順

1. `#{topic}-{連番}` の形式でIDを振る
2. 目的・描くべき要素・注意点を記載
3. `img/{topic}/SOURCES.md` の該当行に番号を紐付ける
