# 電験3種 法規Wiki — テーマページ作成ガイドライン

## 目的・設計思想

> **「条文を暗記する」ではなく「条文を根拠に試験問題を解ける」ようにする**

- 条文は「根拠」であり「主役」ではない
- 試験で狙われる主語・対象・条件・例外・数値・届出先・期限を前面に出す
- B問題（計算）を最優先カテゴリとして扱う
- 試験直前に数値・公式・ひっかけだけ確認できる構造にする

---

## ページメタ情報（必須ヘッダー）

各テーマページ冒頭に以下のメタ情報を表として配置する。

```markdown
| 項目 | 内容 |
|------|------|
| **重要度** | ★★★ / ★★☆ / ★☆☆（出題確率×配点で判定） |
| **頻出度** | 毎回 / 2回に1回 / 散発 |
| **出題形式** | A問題（5択）/ B問題（計算）/ 両方 |
| **対象問題** | R元〜R7 で出題された問題番号（例: A-3, B-14） |
| **関連法令** | 電気事業法○条 / 電技解釈○条 など |
| **最終確認日** | YYYY-MM-DD |
```

**重要度判定基準**: 重要度マトリクス（`knowledge/reference_denken_importance_rule.md` 参照）

---

## 12セクション構成（必須・順序固定）

### 1. まず覚える結論

**目的**: 記憶の骨格を最初に定着させる

- 3〜5行の箇条書きで完結させる
- 数値・条件・例外をここに集約する
- admonition: `!!! success "覚える結論"` で囲む

**禁止**: 「〜が重要です」「〜を理解しましょう」などの曖昧表現

```markdown
!!! success "まず覚える結論"
    - 〇〇の設置義務者は**XXX**（主体を明確に）
    - 届出先は**○○大臣**、期限は**工事開始〇日前**
    - 例外: △△の場合は届出不要
```

---

### 2. 試験で問われること ← **最重要セクション**

**目的**: 問われる構造を先に知り、条文を読む目的を作る

必ず以下の切り口で整理する:

| 切り口 | 具体化の指示 |
|--------|-------------|
| **主語** | 誰が義務を負うか（設置者/工事施工者/主任技術者） |
| **対象** | 何に適用されるか（電圧区分/設備種別） |
| **条件** | 何kV以上、何kW超、工場/一般用 など |
| **例外・除外** | 〇〇の場合は適用除外 |
| **数値** | 具体的な数字（届出期限・日数・距離・電圧） |
| **届出先・申請先** | 経済産業大臣/都道府県知事/産業保安監督部長 |
| **期限** | 事前○日/事後○日/遅滞なく |
| **罰則** | 懲役○年以下/罰金○万円以下 |

```markdown
## 試験で問われること

| 問われる点 | 正解の核心 |
|-----------|-----------|
| 届出義務者 | 電気工作物の設置者（施工者ではない） |
| 届出期限 | 工事開始**30日前**（7日ではない） |
| 届出先 | **経済産業大臣**（都道府県知事ではない） |
| 例外 | 一般用電気工作物は対象外 |
```

---

### 3. 規程・法令原文

**目的**: 条文を「根拠として参照する」ための格納庫

- **原則**: 折りたたみ表示（`??? note "原文"`）
- **必須記載**: 条文番号・出典・確認日
- 長い条文は要点のみ抜粋 → 全文は公式URLに誘導

```markdown
??? note "電気事業法 第○条 （確認日: YYYY-MM-DD）"
    （引用）電気工作物の設置者は、その電気工作物の工事、
    維持及び運用に関する保安を確保するため、……

    **出典**: 電気事業法（昭和39年法律第170号）
    **最終確認**: [e-Gov 法令検索](https://laws.e-gov.go.jp/)
```

---

### 4. かみ砕き解説

**目的**: 初学者が「なぜそうなるか」を理解する

- 専門用語をかみ砕いて説明する（「電気工作物」→「電気で動く設備・機器全体」など）
- 条文の背景にある **理由・目的** を書く（「なぜこの規制があるか」）
- 現場のアナロジーを使う（製造業設備管理者の視点が有効）
- admonition `!!! info` または `!!! tip` を活用

```markdown
!!! tip "かみ砕き解説"
    「自家用電気工作物」とは、電力会社から受電する工場・ビルの設備のこと。
    500kW以上受電する施設が該当し、主任技術者の選任が義務。
    「一般用」と「自家用」の境界線（600V/低圧・高圧）が試験の焦点。
```

---

### 5. 覚える表

**目的**: 数値・区分を横断的に比較記憶させる

- **必須**: マークダウン表形式
- 同じカテゴリの数値を縦に並べる（例: 電圧区分ごとに主任技術者の要件）
- 試験でよく変えてくる「紛らわしい数値」を同一表に配置する

```markdown
## 覚える表

| 区分 | 受電電圧 | 主任技術者資格 | 届出先 |
|------|---------|--------------|--------|
| 一般用 | 600V以下 | 不要 | 不要 |
| 自家用（小） | 600V超〜7,000V | 電験3種以上 | 産業保安監督部長 |
| 自家用（大） | 7,000V超 | 電験2種以上 | 経済産業大臣 |
```

**数値横断表のポイント**:
- 罰則額・届出期限・電圧区分・距離制限などは必ず表化
- 「○○の場合は△△」の例外も表に組み込む

---

### 6. 解き方・判断手順

**目的**: 本番で迷わず解くための思考フローを習得させる

- フローチャートまたはステップ番号で記述
- 「どの情報を見て何を判断するか」を明示する
- B問題（計算）は計算手順をステップで書く

```markdown
## 解き方・判断手順

**A問題（択一）の解法フロー**:

1. 問題文の「主体」を特定（設置者？施工者？）
2. 電気工作物の種別を判定（一般用 / 自家用 / 事業用）
3. 該当する届出義務・資格要件を確認
4. 選択肢の中から「主語・先・期限」が一致するものを選ぶ

**B問題（計算）の解法フロー**:

1. 与えられた値（電圧・電流・力率・受電量）を整理
2. 使う公式を特定
3. 単位変換（kW→W, % → 小数）
4. 計算実行 → 選択肢と照合
```

---

### 7. ひっかけポイント

**目的**: 誤答パターンを事前に潰す

**必須フォーマット**:

```markdown
## ひっかけポイント

| # | × 誤り（ひっかけ選択肢） | ○ 正しい（正解） |
|---|----------------------|----------------|
| 1 | 届出先は**都道府県知事** | 届出先は**経済産業大臣** |
| 2 | 届出期限は工事開始**7日前** | 届出期限は**30日前** |
| 3 | **施工者**が届出義務を持つ | **設置者**が届出義務を持つ |
| 4 | 600V以下は「自家用」 | 600V以下は「一般用」 |
```

**ひっかけの典型パターン**（毎ページ必ず確認）:
- 主語のすり替え（設置者 ↔ 施工者 ↔ 主任技術者）
- 届出先のすり替え（大臣 ↔ 知事 ↔ 監督部長）
- 期限の数字ずらし（30日 ↔ 7日 ↔ 60日）
- 電圧区分の境界値（600V / 7,000V / 170,000V）
- 「以上/超」「以下/未満」の読み替え
- 「しなければならない/することができる（任意）」の混同

### 8. 試験問題・類題

**目的**: 実際の問題形式で理解を確認する

**収録すべき問題タイプ（いずれか2〜3種）**:

```markdown
## 試験問題・類題

### 過去問（要約）
**R5上期 A問題 第3問**
電気事業法の届出義務について、正しいものを選べ。
（選択肢省略、以下「解答解説」参照）

### 自作類題（一問一答）
**Q**: 高圧受電設備（6.6kV受電）の工事を実施する場合、
　　誰がどこに何日前までに届け出なければならないか？

### 正誤問題
以下の記述は○か×か。
1. 電気工作物の設置者は工事開始90日前までに届け出る必要がある。 → （　）
2. 自家用電気工作物の施工者が主任技術者を選任する。 → （　）

### 計算問題（B問題）
需要率60%、設備容量500kW、不等率1.2のとき、最大電力を求めよ。
```

---

### 9. 解答解説

**目的**: 正誤の理由を明確に理解させる

- 「なぜ正しいか」「なぜ誤りか」を両方書く
- 条文の根拠（条番号）を必ず示す
- 計算問題は途中式を全ステップ書く

```markdown
## 解答解説

### 過去問解説
**正解: ②**

① × 工事施工者ではなく**設置者**が届出義務者（電気事業法第48条）
② ○ 設置者が工事開始**30日前**までに経済産業大臣へ届け出る
③ × 都道府県知事ではなく**経済産業大臣**（自家用の場合）
④ × 一般用電気工作物は届出対象外

### 正誤解説
1. × → 正しくは**30日前**（90日は誤り）
2. × → 主任技術者を選任するのは**設置者**（施工者ではない）

### 計算解説
最大電力 = 設備容量 × 需要率 ÷ 不等率
　　　　= 500 × 0.60 ÷ 1.2 = **250 kW**
```

### 10. 1分復習・暗記カード

**目的**: 試験直前の高速確認に使う

- 5問以内の一問一答形式
- 答えを折りたたむ（`??? question`）
- 重要数値・キーワードのみに絞る

```markdown
## 1分復習・暗記カード

??? question "Q1: 自家用電気工作物の届出先は？"
    **経済産業大臣**（または産業保安監督部長へ委任の場合あり）

??? question "Q2: 届出期限は工事開始の何日前か？"
    **30日前**

??? question "Q3: 主任技術者の選任義務者は誰か？"
    **設置者**

??? question "Q4: 600V以下の受電設備は何に分類されるか？"
    **一般用電気工作物**（主任技術者・届出不要）

??? question "Q5: 需要率の公式は？"
    **最大需要電力 ÷ 設備容量 × 100 [%]**
```

## サイト全体の設計原則

### B問題最優先ポリシー

- B問題（計算）はサイドバーで最上位カテゴリに配置
- 需要率・不等率・短絡電流・接地抵抗・変圧器容量の5テーマは「B問題特集」として独立ページ化
- 計算手順ページには必ず「公式一覧表」と「単位変換表」をセット掲載

### 数値横断表（必須）

法規で頻出の数値は以下のカテゴリ別に**サイト横断の早見表**を整備する:

| カテゴリ | 代表的な数値 |
|---------|------------|
| 電圧区分 | 低圧: 直流750V/交流600V以下、高圧: 7kV以下、特別高圧: 7kV超 |
| 届出期限 | 工事計画: 30日前、保安規程: 使用開始前 |
| 離隔距離 | 架空電線と建物: 電圧別に0.3m〜2m |
| 罰則 | 電気事業法違反: 1年以下〜3年以下の懲役など |

### 間違いノート連携

- 各テーマページの「ひっかけポイント」は `strategy/hikkake-patterns.md` に自動転記できる形式で書く
- フォーマット統一: `| テーマ名 | × 誤り | ○ 正しい | 根拠条文 |`

### 試験直前確認モード

試験1週間前に使える「直前確認」セクションを各ページに配置:
- 数値・公式・ひっかけを1画面に収める
- admonition `!!! abstract "直前3分確認"` で統一

## アンチパターン（やってはいけない）

| アンチパターン | 理由 | 代替 |
|-------------|------|------|
| 条文をそのまま羅列する | 何が試験に出るかわからなくなる | 試験で問われることを先に書く |
| 「〜を理解しましょう」で終わる | 学習行動につながらない | 「○○を覚えよ。根拠: △△条」と断言する |
| 過去問番号なしで問題を掲載する | 追跡・正誤確認ができない | 必ず年度・問番号を記載する |
| 全セクションに同量の文章を書く | 重要度が伝わらない | セクション2（試験で問われること）を最長・最詳細にする |
| 法令原文を折りたたまない | 読みづらくなる | ??? note で折りたたむ |
| 計算問題に途中式を書かない | 正解の根拠が不明 | 全ステップを記載する |
| § 記号を使う | 文字化けリスク | 1. 2. 形式に統一 |
