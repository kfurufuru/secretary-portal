# ERRORS.md

**運用ルール**: `.claude/rules/work-rules.md` §9 に従い、同じ修正で2回以上失敗した場合、または原因調査に時間がかかった場合に追記する。

軽微な試行錯誤や1回で解決した問題は記録しない。

## 記録テンプレート

```markdown
## YYYY-MM-DD: <短い件名>

### 何がうまくいかなかったか
- 症状・エラーメッセージ・観測された挙動

### 原因
- 真因（推測ではなく確認できた根拠ベース）

### 最終的にうまくいった方法
- 実施した手順・コマンド・修正内容

### 次回同じ失敗を避けるための注意点
- チェックリスト化できる学び
- 関連 feedback_*.md / memory リンクがあれば追記
```
